# 从零开始学 OpenAI Codex CLI

这是一个 5 阶段实战学习路径，帮助你从零开始系统学习 OpenAI Codex CLI。它既是面向初学者的 OpenAI Codex 教程，也是可落地的 Codex Agent 指南，适合想要掌握 Codex CLI、AI 编程代理和终端自动化工作流的学习者。

## 为什么学习这门课

OpenAI Codex CLI 可以把终端变成 AI 编程工作台。本课程从第一次启动开始，逐步带你理解项目配置、记忆、审批模式、Hooks、MCP、Skills、Subagents、Worktrees，以及如何构建自己的 CLI 工具。

你可以用这个仓库来：

- 在真实项目中使用前，先理解 Codex CLI 的能力边界。
- 按阶段学习配置、记忆、审批、扩展和高级代理工作流。
- 通过任务驱动练习，而不是零散阅读资料。
- 更安全、更高效地使用 AI 编程代理。

## 你将学到什么

| 阶段 | 路径 | 学习成果 |
| --- | --- | --- |
| 1 | [`stages/stage-01-meet-codex`](stages/stage-01-meet-codex/) | 认识 Codex CLI，运行基础流程，理解审批模式。 |
| 2 | [`stages/stage-02-project-config`](stages/stage-02-project-config/) | 学会项目配置、记忆、CLI 设置和可复用上下文。 |
| 3 | [`stages/stage-03-extend-codex`](stages/stage-03-extend-codex/) | 使用 Skills、Hooks、MCP 和自动化能力扩展 Codex。 |
| 4 | [`stages/stage-04-advanced`](stages/stage-04-advanced/) | 掌握高级代理模式、Subagents、Worktrees 和更安全的工作流。 |
| 5 | [`stages/stage-05-build-cli-tool`](stages/stage-05-build-cli-tool/) | 通过构建 CLI 工具，把 Codex CLI 实践串联起来。 |

补充知识文档位于 [`knowledge/`](knowledge/)。

## 前置要求

- 熟悉基础命令行操作。
- 了解 Git 和 GitHub 基础。
- 已安装并可使用 OpenAI Codex CLI。
- 有代码编辑器和终端环境。
- 对 AI 编程代理与自动化工作流有兴趣。

## 快速开始

1. 克隆仓库：

   ```bash
   git clone https://github.com/aiflowlearn/zero2codex.git
   cd zero2codex
   ```

2. 从第 1 阶段开始：

   ```bash
   open stages/stage-01-meet-codex/README.md
   ```

3. 阅读阶段 README，然后完成任务：

   ```bash
   open stages/stage-01-meet-codex/tasks.md
   ```

4. 按顺序完成全部 5 个阶段。

## 在线练习

建议把本仓库当作你的练习地图：

- 先阅读每个阶段的 README 和概念内容。
- 在沙盒项目或一次性项目中完成 `tasks.md`。
- 把有用的命令、提示词和配置模式记录到自己的笔记里。
- 对较难任务反复练习，直到你能解释 Codex CLI 正在做什么以及为什么这样做。

如果你发布学习笔记，欢迎链接回本课程，帮助更多学习者使用同一条路径。

## FAQ

### 这门课适合初学者吗？

适合。本课程面向 Codex CLI 初学者，但默认你已经会使用终端和 Git。

### 这是 OpenAI 官方文档吗？

不是。这是 AIFlowLearn 的独立学习路径。涉及产品行为时，请以 OpenAI Codex CLI 最新官方文档为准。

### 可以用于工作坊或学习小组吗？

可以，但必须遵守 CC BY-NC-SA 4.0 协议，仅限非商业学习使用，改编内容也需要使用相同协议。

### 这个仓库包含工具脚手架吗？

不包含。本仓库只包含课程内容和知识文档。
